# Keychron Q11 — Hardware Design Files

> Split 75% custom mechanical keyboard for adjustable ergonomic positioning and programmable workflows.

**Product page:** [Keychron Q11](https://www.keychron.com/products/keychron-q11-qmk-custom-mechanical-keyboard)

![Q11](https://i.imgur.com/FPTNqKj.jpg)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% Split |
| Size | 84-key |
| Case Material | CNC aluminum |
| Connectivity | USB-C wired |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/master/keyboards/keychron/q11) |
