# Keychron Q9 — Hardware Design Files

> 40% compact custom mechanical keyboard designed for minimal layouts and heavy programmability.

**Product page:** [Keychron Q9](https://www.keychron.com/products/keychron-q9-qmk-custom-mechanical-keyboard)

![Q9](https://i.imgur.com/G8xJ4TRh.jpg)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 40% |
| Size | 52-key |
| Case Material | CNC aluminum |
| Connectivity | USB-C wired |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/master/keyboards/keychron/q9) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications, where applicable |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |
