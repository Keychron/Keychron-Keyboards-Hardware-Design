# Keychron Q12 — Hardware Design Files

> Southpaw 96% custom mechanical keyboard with the numpad shifted to the left for mouse space.

**Product page:** [Keychron Q12](https://www.keychron.com/products/keychron-q12-qmk-custom-mechanical-keyboard)

![Keychron Q12](https://cdn.shopify.com/s/files/1/0059/0630/1017/t/5/assets/keychronq12qmkcustommechanicalkeyboard1-1676101752350.jpg?v=1676101755)

## Specifications

| Spec | Detail |
|---|---|
| Layout | Southpaw 96% |
| Size | 100-key |
| Case Material | CNC aluminum |
| Connectivity | USB-C wired |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/master/keyboards/keychron/q12) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications, where applicable |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |
