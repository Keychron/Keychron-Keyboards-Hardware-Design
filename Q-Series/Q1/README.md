# Keychron Q1 — Hardware Design Files

> 75% layout, 82-key gasket-mounted aluminum custom mechanical keyboard with rotary encoder knob.

**Product page:** [Keychron Q1](https://www.keychron.com/products/keychron-q1)

![Keychron Q1](https://cdn.shopify.com/s/files/1/0059/0630/1017/t/5/assets/keychronq1v2custommechanicalkeyboard15-1658373808819.jpg?v=1658373894)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% (82-key) |
| Case Material | CNC aluminum |
| Mount | Gasket |
| Knob | Yes (rotary encoder) |
| Connectivity | USB-C wired |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/master/keyboards/keychron/q1) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |
