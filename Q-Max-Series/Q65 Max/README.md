# Keychron Q65 Max - Hardware Design Files

> 65% full-metal QMK/VIA wireless custom mechanical keyboard with a programmable knob, tri-mode connectivity, and acoustic foams.

**Product page:** [Keychron Q65 Max](https://www.keychron.com/products/keychron-q65-max-qmk-via-wireless-custom-mechanical-keyboard)

![Q65 Max](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron-Q65-Max-QMK-VIA-Custom-Mechanical-Keyboard-1.jpg)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 65% |
| Body Material | Full CNC aluminum |
| Mount | Double-gasket |
| Knob | Yes |
| Connectivity | 2.4 GHz / Bluetooth 5.1 / USB-C wired |
| Polling Rate | 1000 Hz (2.4 GHz and wired) / 90 Hz (Bluetooth) |
| Switch Support | Hot-swappable 3-pin and 5-pin MX mechanical switches |
| Keycaps | KSA double-shot PBT |
| Firmware | QMK |
