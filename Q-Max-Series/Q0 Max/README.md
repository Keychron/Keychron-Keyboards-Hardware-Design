# Keychron Q0 Max — Hardware Design Files

> Full-metal wireless custom number pad with a macro column, left-side knob, QMK/VIA support, and tri-mode connectivity.

**Product page:** [Keychron Q0 Max](https://www.keychron.com/products/keychron-q0-max-qmk-custom-number-pad)

![Q0 Max](https://www.keychron.com/cdn/shop/files/Q0-Max-Iconic-Features.jpg?crop=center&height=1200&v=1754020491&width=1200)

## Current Repository Status

- This folder is ready for future hardware design uploads for the Q0 Max.
- No local `stp`, `dwg`, `dxf`, `pdf`, or related CAD assets are included yet.
