# Keychron Q2 Max — Hardware Design Files

> 65% full metal QMK/VIA wireless custom mechanical keyboard with tri-mode connectivity and 1,000 Hz polling.

- This folder is ready for future hardware design uploads for the Q2 Max.
- No local `stp`, `dwg`, `dxf`, `pdf`, or related CAD assets are included yet.
