# Keychron Q60 Max - Hardware Design Files

> 60% full-metal QMK/VIA wireless custom mechanical keyboard with tri-mode connectivity and 1000 Hz polling.

**Product page:** [Keychron Q60 Max](https://www.keychron.com/products/keychron-q60-max-qmk-via-wireless-custom-mechanical-keyboard)

![Q60 Max](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron-Q60-Max-QMK-VIA-Custom-Mechanical-Keyboard-1.jpg?v=1701933718)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 60% |
| Body Material | Full CNC aluminum |
| Mount | Double-gasket |
| Connectivity | 2.4 GHz / Bluetooth 5.1 / USB-C wired |
| Polling Rate | 1000 Hz (2.4 GHz and wired) / 90 Hz (Bluetooth) |
| Switch Support | Hot-swappable 3-pin and 5-pin MX mechanical switches |
| Keycaps | Cherry dye-sub PBT |
| Firmware | QMK |
