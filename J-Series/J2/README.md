# Keychron J2 - Hardware Design Files

> 75% QMK wireless mechanical keyboard with 2.4 GHz, Bluetooth, and USB-C wired connectivity.

**Product page:** [Keychron J2 QMK Wireless Mechanical Keyboard](https://www.keychron.com/products/keychron-j2-qmk-wireless-mechanical-keyboard)

![Keychron J2](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron-J2-QMK-Wireless-Mechanical-Keyboard-1.jpg?v=1757405687)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% |
| Connectivity | 2.4 GHz / Bluetooth / USB-C wired |
| Firmware | QMK / Keychron Launcher |
| Hot-swap | Yes, depending on version |
| Case Material | ABS plastic |
| Keycaps | Double-shot PBT |
