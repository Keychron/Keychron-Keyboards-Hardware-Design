# Keychron J7 - Hardware Design Files

> Note: The files in this folder are applicable to Z7 Max.

## Included Files

| File | Description |
|---|---|
| `J7-Aluminum-Bottom-Case.stp` | Aluminum bottom case 3D CAD model |
| `J7-LSA-US-Full-Model-20260305.stp` | US full keyboard 3D CAD model with LSA profile keycaps |
| `J7-US-Aluminum-Top-Case.stp` | US aluminum top case 3D CAD model |
| `J7-US-PCB-20260526.dwg` | US PCB 2D CAD drawing |
| `J7_US_Plate_20260526.dwg` | US plate 2D CAD drawing |
| `J7_US_Plate_20260526.dxf` | US plate manufacturing-ready 2D CAD drawing |
| `J7_US_Plate_20260526.pdf` | US plate reference drawing |

| Format | Free Software | Commercial Software |

| `.dwg` | LibreCAD (limited), QCAD | AutoCAD, DraftSight |
