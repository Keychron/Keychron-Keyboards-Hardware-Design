# Keychron G4 - Hardware Design Files

> Ergonomic wireless optical mouse with PixArt 3950 sensor, 8K polling, tri-mode connectivity, and a lightweight 55 g body.

**Product page:** [Keychron G4 Wireless Mouse](https://www.keychron.com/collections/mice-collection/products/keychron-g4-wireless-mouse)

![Keychron G4](http://www.keychron.com/cdn/shop/files/Keychron-G4-Wireless-Mouse-Features_2.jpg?crop=center&height=1200&v=1776248742&width=1200)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless optical mouse |
| Shape | Ergonomic |
| Weight | 55 g |
| Sensor | PixArt 3950 |
| DPI | 100 - 30000 |
| Polling Rate | Up to 8000 Hz (2.4 GHz / wired) |
| Connectivity | 2.4 GHz / Bluetooth 5.3 / USB-C wired |
| Switch | Huano micro switch |
| Battery | 500 mAh |
| Body Material | ABS + PC |
