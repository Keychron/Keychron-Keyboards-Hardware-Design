# Keychron M2 Mini — Hardware Design Files

> Compact ultra-light wireless optical mouse with ergonomic shaping, tri-mode connectivity, and up to 8K polling depending on version.

**Product page:** [Keychron M2 Mini Wireless Mouse](https://www.keychron.com/products/keychron-m2-mini-wireless-mouse)

![Keychron M2 Mini](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron-M2-mini-wireless-optical-mouse-6.jpg?v=1697444925)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless optical mouse |
| Shape | Ergonomic right-handed compact |
| Weight | 50 g |
| Sensor | PixArt 3395 / PixArt 3950 (version-dependent) |
| DPI | Up to 30000 |
| Polling Rate | Up to 8000 Hz (2.4 GHz / wired), 125 Hz (Bluetooth) |
| Connectivity | Bluetooth / 2.4 GHz / USB-C wired |
| Switch | Huano 80M micro switch |
| Battery | 600 mAh |
| Body Material | ABS |
