# Keychron BM25 - Hardware Design Files

> Quiet wireless office mouse with PAW3212 sensor, tri-mode connectivity, and lightweight ergonomic design.

**Product page:** [Keychron BM25 Wireless Mouse](https://www.keychron.com/products/keychron-bm25-wireless-mouse)

![Keychron BM25](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron-BM25-Mouse-1.jpg?v=1753416918)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless office mouse |
| Shape | Ergonomic |
| Weight | 79 +- 3 g |
| Sensor | PAW3212 |
| DPI | 100 - 2400 |
| IPS | 30 |
| Polling Rate | Max 1000 Hz (2.4 GHz / wired), Max 125 Hz (Bluetooth) |
| Connectivity | Bluetooth 5.2 / 2.4 GHz / USB-C wired |
| Switch | Huano micro switch |
| Battery | 600 mAh |
| Body Material | ABS |
