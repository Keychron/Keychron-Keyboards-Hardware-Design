# Keychron M4 Hex Light Mod - Community Files

> Customer modification files for the Keychron M4, designed by Reddit customer SlayCC and shared separately from the official Keychron M4 hardware design files.

![Keychron M4 Hex Light Mod by SlayCC](https://media.printables.com/media/prints/7b2608d4-9fbd-4cb2-bbe7-a3c2af2d4d19/images/13040483_0ba6466d-49e8-4d9b-8879-4c443572050c_817a8b9b-048a-44fb-bb3a-f3696dbe63ec/img_20260531_003300302.jpg)

## Mod Summary

Mod of the Keychron M4 fingertip mouse at 12 grams and 51 mm width.
