# Keychron K8 - Hardware Design Files

> Tenkeyless wireless mechanical keyboard with Bluetooth 5.1, USB-C wired mode, Mac/Windows support, and optional hot-swappable switches.

**Product page:** [Keychron K8 Tenkeyless Wireless Mechanical Keyboard](https://www.keychron.com/products/keychron-k8-tenkeyless-wireless-mechanical-keyboard)

![Keychron K8](https://cdn.shopify.com/s/files/1/0059/0630/1017/t/5/assets/keychronk8wirelessmechanicalkeyboard1-1646210675142.jpg?v=1646210677)

## Specifications

| Spec | Detail |
|---|---|
| Layout | TKL / 80% / ANSI |
| Number of Keys | 87 |
| Switch Type | Gateron G Pro Mechanical / Keychron Super Mechanical / Keychron Optical |
| Frame Material | ABS / ABS + aluminum frame |
| Keycap Material | ABS |
| Keycap Profile | OEM |
| Backlight | White backlight / RGB backlight / RGB backlight aluminum frame |
| Connectivity | Bluetooth 5.1 / USB-C wired |
| Polling Rate | 1000 Hz (wired) / 90 Hz (Bluetooth) |
| Battery | 4000 mAh rechargeable li-polymer |
| Wireless Working Time | Up to 240 hours (backlight off) |
| Compatible System | macOS / Windows |
