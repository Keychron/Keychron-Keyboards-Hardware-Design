# Keychron K3 - Hardware Design Files

> Ultra-slim 75% wireless mechanical keyboard with Bluetooth 5.1, USB-C wired mode, and support for macOS and Windows.

**Product page:** [Keychron K3 Wireless Mechanical Keyboard](https://www.keychron.com/products/keychron-k3-wireless-mechanical-keyboard)

![Keychron K3](https://www.keychron.com/cdn/shop/products/Keychron-K3-ultra-slim-wireless-mechanical-keyboard--5.jpg?v=1634644048)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% |
| Number of Keys | 84 |
| Profile | Low-profile |
| Switch Type | Low Profile Gateron Mechanical / Low Profile Keychron Optical (Hot-swappable) |
| Backlight | White backlight / RGB backlight |
| Connectivity | Bluetooth 5.1 / USB-C wired |
| Hot-swappable | Available on the low profile Keychron Optical version |
| Compatible System | macOS / Windows |
