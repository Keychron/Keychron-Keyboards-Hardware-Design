# Keychron Q2 HE — Hardware Design Files

> 65% magnetic switch custom keyboard with a programmable knob, adjustable actuation, rapid trigger, and tri-mode connectivity.

**Product page:** [Keychron Q2 HE](https://www.keychron.com/products/keychron-q2-he-qmk-wireless-custom-keyboard)

![Q2 HE](https://www.keychron.com/cdn/shop/files/Q2-HE-Iconic-Features.jpg?crop=center&height=1200&v=1754886504&width=1200)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 65% |
| Case Material | Full metal |
| Knob | Yes |
| Switch Type | Magnetic / Hall Effect |
| Connectivity | 2.4 GHz / Bluetooth / USB-C |
| Firmware | Keychron Launcher / QMK |
