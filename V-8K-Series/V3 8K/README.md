# Keychron V3 8K — Hardware Design Files

> 80% wired mechanical keyboard with QMK support, PBT keycaps, and 8K polling.

**Product page:** [Keychron V3 8K](https://www.keychron.com/products/keychron-v3-8k-qmk-wired-custom-mechanical-keyboard)

![V3 8K](https://www.keychron.com/cdn/shop/files/Keychron-V3-8K-Features.jpg?crop=center&height=1200&v=1773105036&width=1200)

## Current Repository Status

- This folder is ready for future hardware design uploads for the V3 8K.
- No local `stp`, `dwg`, `dxf`, `pdf`, or related CAD assets are included yet.

## Expected File Types

| File Type | Typical Use |
|---|---|
| `stp` | Case, plate, or full assembly 3D CAD |
| `dwg` / `dxf` | Plate drawings and 2D manufacturing references |
| `pdf` | Reference drawings and quick-view documents |

## Product Snapshot

| Spec | Detail |
|---|---|
| Layout | 80% |
| Connectivity | Wired |
| Polling Rate | 8,000 Hz |
| Keycaps | PBT |
| Firmware | QMK |
| Knob | Yes |
