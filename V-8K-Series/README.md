# V 8K Series — Hardware Design Files

> Wired V-series keyboards with QMK support, 8K polling, and folders ready for future CAD releases.

Part of the [Keychron Keyboards Hardware Design](https://github.com/Keychron/Keychron-Keyboards-Hardware-Design) repository.

## Available Models

| Model | Layout | Size | Knob | Firmware |
|---|---|---|---|---|
| [V1 8K](./V1%208K/README.md) | 75% | Compact | Yes | QMK |
| [V3 8K](./V3%208K/README.md) | 80% | TKL | Yes | QMK |
| [V5 8K](./V5%208K/README.md) | 96% | Compact | Yes | QMK |
| [V6 8K](./V6%208K/README.md) | 100% | Full-size | Yes | QMK |

## Repository Contents

This series folder currently includes model folders for:

- [V1 8K](./V1%208K/README.md)
- [V3 8K](./V3%208K/README.md)
- [V5 8K](./V5%208K/README.md)
- [V6 8K](./V6%208K/README.md)

The folders are ready for future `stp`, `dwg`, `dxf`, `pdf`, or related hardware uploads.

## Download Files

| File | Description | Format |
|---|---|---|
| **Future CAD assets** | Placeholder folders ready for case, plate, or full-model releases | STP / DWG / DXF / PDF |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.stp` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dwg` | LibreCAD (limited), QCAD | AutoCAD, DraftSight |
| `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
