# Keychron Q13 Pro — Hardware Design Files

> Alice 96% wireless custom mechanical keyboard with a programmable knob, Bluetooth 5.1, and USB-C wired connectivity.

**Product page:** [Keychron Q13 Pro](https://www.keychron.com/products/keychron-q13-pro-alice-layout-qmk-via-wireless-custom-mechanical-keyboard)

![Keychron Q13 Pro](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Q13-Pro-3.jpg?v=1694157107)

## Specifications

| Spec | Detail |
|---|---|
| Layout | Alice / 96% (96-key) |
| Mount | Double-gasket |
| Case Material | Full aluminum |
| Knob | Yes |
| Connectivity | Bluetooth 5.1 / USB-C |
| Polling Rate | 1000 Hz (wired) / 90 Hz (wireless) |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/bluetooth_playground/keyboards/keychron/q13_pro) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with Alice layout cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |
