# Lemokey P3 HE — Hardware Design Files

> 80% wireless magnetic switch custom gaming keyboard with a knob, gasket structure, advanced acoustic foam, and double-shot PBT keycaps.

**Product page:** [Lemokey P3 HE](https://www.keychron.com/products/lemokey-p3-he-wireless-magnetic-switch-custom-gaming-keyboard)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 80% |
| Knob | Yes |
| Switch Type | Magnetic / Hall Effect |
| Connectivity | 2.4 GHz / Bluetooth / USB-C |
| Firmware | Keychron Launcher / QMK |
