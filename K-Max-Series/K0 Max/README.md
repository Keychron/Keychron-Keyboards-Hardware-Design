# Keychron K0 Max — Hardware Design Files

> Wireless custom number pad files for the K0 Max, including enclosure parts, plate geometry, knob, keycap, stabilizer, and a full-model assembly.

**Product page:** [Keychron K0 Max](https://www.keychron.com/products/keychron-k0-max-qmk-wireless-custom-number-pad)

## Specifications

| Spec | Detail |
|---|---|
| Layout | Number pad |
| Profile | Standard |
| Knob | Yes |
| Connectivity | Bluetooth / 2.4 GHz / USB-C |
| Firmware | QMK / VIA |

## Included Files

| File | Description |
|---|---|
| `K0-Max-US-Bottom-Case.stp` | Bottom case enclosure |
| `K0-Max-US-Aluminum-Plate.dwg` | US aluminum plate drawing |
| `K0-Max-US-Aluminum-Plate.pdf` | US aluminum plate reference sheet |
| `K0-Max-US-Aluminum-Plate.stp` | 3D plate model |
| `K0-Max-US-Knob.stp` | Rotary encoder knob model |
| `K0-Max-US-LSA-Keycap.stp` | LSA-profile keycap model |
| `K0-Max-Stabilizer-24mm-3D.stp` | Stabilizer model |
| `K0-Max-US-Full-Model-20251031.stp` | Complete assembled model |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.stp` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dwg` | LibreCAD (limited), QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
