# Keychron V3 - Hardware Design Files

> 80% wired QMK custom mechanical keyboard with ABS case, hot-swappable PCB, PBT keycaps, and optional programmable knob.

**Product page:** [Keychron V3 QMK Custom Mechanical Keyboard](https://www.keychron.com/products/keychron-v3-qmk-custom-mechanical-keyboard)

![Keychron V3](https://cdn.shopify.com/s/files/1/0059/0630/1017/t/5/assets/keychronv3custommechanicalkeyboardwithqmkviacarbonblackversion-1665384169419.jpg?v=1665384171)

## Specifications

| Spec | Detail |
|---|---|
| Layout | TKL / 80% |
| Body Material | ABS plastic |
| Plate Material | Steel |
| Mount | Tray mount |
| Connectivity | USB-C wired |
| Polling Rate | 1000 Hz |
| Switch Support | Hot-swappable 3-pin and 5-pin MX mechanical switches |
| Keycaps | Double-shot OSA PBT |
| Knob | Optional, depending on version |
| Firmware | QMK / Keychron Launcher |
