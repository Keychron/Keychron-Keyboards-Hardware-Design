# Keychron V2 QMK - Hardware Design Files

> 65% wired QMK custom mechanical keyboard with ABS case, hot-swappable PCB, PBT keycaps, and optional programmable knob.

**Product page:** [Keychron V2 QMK Custom Mechanical Keyboard](https://www.keychron.com/products/keychron-v2-qmk-custom-mechanical-keyboard)

![Keychron V2 QMK](https://cdn.shopify.com/s/files/1/0059/0630/1017/t/5/assets/keychron-v2-custom-mechanical-keyboard-with-qmk-via-1663401102043.jpg?v=1663401106)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 65% |
| Body Material | ABS plastic |
| Plate Material | Steel |
| Mount | Tray mount |
| Connectivity | USB-C wired |
| Polling Rate | 1000 Hz |
| Switch Support | Hot-swappable 3-pin and 5-pin MX mechanical switches |
| Keycaps | Double-shot OSA PBT |
| Knob | Optional, depending on version |
| Firmware | QMK / Keychron Launcher |
