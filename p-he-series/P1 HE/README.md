# Lemokey P1 HE — Hardware Design Files

> 75% wireless magnetic switch gaming keyboard with rapid trigger, adjustable actuation, tri-mode connectivity, and a programmable knob.

**Product page:** [Lemokey P1 HE](https://www.keychron.com/products/lemokey-p1-he-wireless-magnetic-switch-custom-gaming-keyboard)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% (81-key) |
| Mount | Gasket |
| Case Material | Full metal |
| Knob | Yes |
| Switch Type | Gateron Double-Rail Magnetic Switch |
| Connectivity | Bluetooth 5.2 / 2.4 GHz / USB-C |
| Polling Rate | 1000 Hz (2.4 GHz and wired) |
| Rapid Trigger | Yes |
| Adjustable Actuation | 0.2 - 3.8 mm |
| Firmware | QMK / Lemokey Launcher / VIA |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |
