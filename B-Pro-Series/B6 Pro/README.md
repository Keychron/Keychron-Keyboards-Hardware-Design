# Keychron B6 Pro - Hardware Design Files

> Ultra-slim full-size wireless keyboard with 2.4 GHz, Bluetooth, wired connectivity, and Keychron Launcher customization.

**Product page:** [Keychron B6 Pro Ultra-Slim Wireless Keyboard](https://www.keychron.com/products/keychron-b6-pro-ultra-slim-wireless-keyboard)

![Keychron B6 Pro](http://www.keychron.com/cdn/shop/files/B6-Pro-Iconic-Features.jpg?crop=center&height=1200&v=1761279632&width=1200)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 100% / full size |
| Number of Keys | 109 |
| Body Material | ABS |
| Keycap Material | ABS |
| Backlight | Non-backlit |
| Connectivity | 2.4 GHz / Bluetooth 5.2 / USB-C wired |
| Polling Rate | 1000 Hz (2.4 GHz and wired) / 90 Hz (Bluetooth) |
| Battery | 800 mAh rechargeable li-polymer |
| Wireless Working Time | Up to 8 months |
| Firmware | Keychron Launcher / ZMK |
