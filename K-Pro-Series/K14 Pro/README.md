# Keychron K14 Pro — Hardware Design Files

> 70% wireless mechanical keyboard with QMK/VIA support, Bluetooth 5.1, and USB-C wired connectivity.

**Product page:** [Keychron K14 Pro](https://www.keychron.com/products/keychron-k14-pro-qmk-via-wireless-mechanical-keyboard)

![K14 Pro](https://i.imgur.com/oDkNCbI.jpg)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 70% (72-key) |
| Profile | Standard |
| Case Material | ABS / ABS + Aluminum |
| Knob | No |
| Connectivity | Bluetooth 5.1 / USB-C |
| Polling Rate | 1000 Hz (wired) |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/bluetooth_playground/keyboards/keychron/k14_pro) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |
