# Keychron K4 HE — Hardware Design Files

> 96% 100-key wireless magnetic switch keyboard with adjustable actuation, rapid trigger support, and tri-mode connectivity.

**Product page:** [Keychron K4 HE](https://www.keychron.com/products/keychron-k4-he-wireless-magnetic-switch-custom-keyboard)

![Keychron K4 HE](https://www.keychron.com/cdn/shop/files/K4-HE-Iconic-Features.jpg?crop=center&height=1200&v=1758336727&width=1200)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 96% (100-key) |
| Versions | Standard Version / Special Edition |
| Case Material | ABS + Aluminum (Standard) / ABS + Aluminum + Wood (Special Edition) |
| Switch Type | Gateron Double-Rail Magnetic Switch |
| Connectivity | Bluetooth 5.2 / 2.4 GHz / USB-C |
| Polling Rate | 1000 Hz |
| Adjustable Actuation | Yes |
| Firmware | QMK / Keychron Launcher |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/hall_effect_playground/keyboards/keychron/k4_he) |
