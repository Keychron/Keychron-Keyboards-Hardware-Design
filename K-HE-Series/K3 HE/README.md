# Keychron K3 HE — Hardware Design Files

> Ultra-slim 75% wireless magnetic switch keyboard with low-profile Hall Effect switches, rapid trigger support, and tri-mode connectivity.

**Product page:** [Keychron K3 HE](https://www.keychron.com/products/keychron-k3-he-wireless-magnetic-switch-custom-keyboard)

![Keychron K3 HE](https://www.keychron.com/cdn/shop/files/Keychron-K3-HE-Features.jpg?crop=center&height=1200&v=1774250330&width=1200)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% compact slim |
| Versions | Fully Assembled / Fully Assembled ISO |
| Body Material | Aluminum + Wood |
| Bottom Case Material | ABS Plastic |
| Switch Type | Keychron Ultra-Fast Lime Low-Profile Magnetic Switch |
| Keycap | LSA double-shot PBT low-profile keycaps |
| Connectivity | Bluetooth 5.2 / 2.4 GHz / USB-C |
| Polling Rate | 1000 Hz (wired and 2.4 GHz) / 125 Hz (Bluetooth) |
| Adjustable Actuation | 0.2-2.9 mm |
| Rapid Trigger | Yes |
| Firmware | QMK / Keychron Launcher |
