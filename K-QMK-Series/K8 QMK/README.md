# Keychron K8 QMK — Hardware Design Files

> TKL K-series QMK keyboard with Bluetooth 5.2 and wired connectivity.

**Product page:** [Keychron K8 QMK Wireless Mechanical Keyboard Version 2](https://www.keychron.com/products/keychron-k8-qmk-wireless-mechanical-keyboard-version-2)

![Keychron K8 QMK](https://www.keychron.com/cdn/shop/files/K8X-Iconic-Features.jpg?crop=center&height=1200&v=1764064043&width=1200)

## Specifications

| Spec | Detail |
|---|---|
| Layout | TKL / 80% |
| Size | 87-key |
| Profile | Standard |
| Knob | — |
| Connectivity | Bluetooth 5.2 / USB-C |
| Hot-swap | Yes |
| Firmware | QMK / Keychron Launcher |

## Repository Status

CAD files for this model have not been added to this repository yet. This folder is prepared so related hardware design files can be dropped in later.

## Planned File Types

| File Type | Purpose |
|---|---|
| `stp` / `step` | Case and assembly 3D models |
| `dwg` | 2D plate and mechanical drawings |
| `dxf` | Manufacturing-ready 2D geometry when available |
| `pdf` | Reference drawings and dimension sheets |
