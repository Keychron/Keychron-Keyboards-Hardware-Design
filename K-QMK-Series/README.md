## Download Files

| File | Description | Format |
|---|---|---|
| **Case** | Keyboard enclosure parts | STEP |
| **Plate** | Switch mounting plate with cutout dimensions | DWG / DXF / PDF |
| **Keycap** | Keycap 3D model | STEP |
| **Stabilizer** | Stabilizer hardware models | STEP |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.stp` | FreeCAD, Onshape (browser) | Fusion 360, SolidWorks |
| `.dwg` | LibreCAD (limited), QCAD | AutoCAD, DraftSight |
| `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
