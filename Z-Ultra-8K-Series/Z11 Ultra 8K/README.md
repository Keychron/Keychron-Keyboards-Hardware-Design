# Keychron Z11 Ultra 8K - Hardware Design Files

> Split wireless custom mechanical keyboard hardware design files for the Keychron Z11 Ultra 8K.

![Z11 Ultra 8K](./z11-ultra-8k-product.jpg)

## Included Files

| File | Description |
|---|---|
| `Z11_Ultra_8K_US_Full_Model-20260616.stp` | US full keyboard 3D CAD model |
| `Z11_Ultra_8K_US_Left Bottom Case_Aluminum-20260616.stp` | US left bottom aluminum case 3D CAD model |
| `Z11_Ultra_8K_US_Left Top Case_Aluminum-20260616.stp` | US left top aluminum case 3D CAD model |
| `Z11_Ultra_8K_US_PCB-20250902.dwg` | US PCB 2D CAD drawing |
| `Z11_Ultra_8K_US_Right Bottom Case_Aluminum-20260616.stp` | US right bottom aluminum case 3D CAD model |
| `Z11_Ultra_8K_US_Right Top Case_Aluminum-20260616.stp` | US right top aluminum case 3D CAD model |
| `Z11_Ultra_US_Plate_Of_Left and Right Case-20260616.dwg` | US left and right case plate 2D CAD drawing |
| `Z11_Ultra_US_Plate_Of_Left and Right Case-20260616.pdf` | US left and right case plate reference drawing |
