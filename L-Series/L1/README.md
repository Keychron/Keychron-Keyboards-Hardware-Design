# Lemokey L1

> **Product page:** [Lemokey L1 QMK/VIA Wireless Custom Gaming Keyboard](https://www.keychron.com/products/lemokey-l1-qmk-via-wireless-custom-gaming-keyboard)

A 75% layout wireless custom gaming mechanical keyboard with full aluminum body.

## Overview

The Lemokey L1 is a 75% layout gasket-mounted aluminum keyboard featuring 2.4 GHz, Bluetooth 5.1, and USB-C wired connectivity. Built with a 6063 aluminum frame through 24 processes including CNC machining, polishing, anodizing, and sandblasting, with a programmable rotary encoder knob.

## Specifications

- **Layout:** 75%, compact
- **Mount:** Double-gasket mount
- **Case:** CNC-machined 6063 aluminum
- **Connectivity:** 2.4 GHz / Bluetooth 5.1 / USB-C wired
- **Hot-swap:** Yes (supports 3-pin and 5-pin switches)
- **Firmware:** QMK / VIA supported
- **Switches:** Gateron Jupiter (fully assembled version)
- **Keycaps:** Double-shot Cherry PBT
- **Backlight:** South-facing RGB LED
