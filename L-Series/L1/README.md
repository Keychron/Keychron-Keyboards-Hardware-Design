# Lemokey L1 — Hardware Design Files

> 75% layout wireless custom gaming mechanical keyboard with full CNC aluminum body and rotary knob.

**Product page:** [Lemokey L1](https://www.keychron.com/products/lemokey-l1-qmk-via-wireless-custom-gaming-keyboard)

![Lemokey L1](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Lemokey-L1-black.jpg?v=1710125438)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% (compact) |
| Size | 82-key |
| Mount | Double-gasket |
| Case Material | CNC-machined 6063 aluminum |
| Knob | Yes |
| Connectivity | Bluetooth 5.1 / 2.4 GHz / USB-C |
| Hot-swap | Yes |
| Firmware | QMK / VIA |

## Included Files

| File | Description |
|---|---|
| `L1-Bottom-Top-Case.stp` | Combined top and bottom enclosure model |
| `L1-Top-Case.stp` | Top case only |
| `L1-Knob.stp` | Rotary encoder knob |
| `L1-US-Full-Model-20231230.stp.zip` | Complete assembled model (compressed) |
| `L1-Plate-File.dxf` | Switch plate cutout drawing |
| `L1-Plate-File.pdf` | Switch plate reference (PDF) |
| `Stabilizer-2.25u.stp` | 2.25u stabilizer model |
| `Stabilizer-6.25u.stp` | 6.25u stabilizer model |
