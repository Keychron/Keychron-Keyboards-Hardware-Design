# Keychron K3 Ultra 8K - Hardware Design Files

> Ultra-slim 75% wireless custom mechanical keyboard with 2.4 GHz, Bluetooth, and wired connectivity, ZMK firmware, Keychron Launcher customization, an 8K polling rate, and up to 550 hours of battery life.

**Product page:** [Keychron K3 Ultra 8K](https://www.keychron.com/products/keychron-k3-ultra-8k-wireless-custom-mechanical-keyboard)

![K3 Ultra 8K](https://www.keychron.com/cdn/shop/files/Keychron-K3-Ultra-8K-Wireless-Custom-MechanicalKeyboard-1.jpg?v=1774260926&width=1000)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% compact slim |
| Versions | Fully Assembled / Fully Assembled ISO |
| Body Material | ABS + Wood |
| Bottom Case Material | ABS Plastic |
| Switch Type | Keychron Milk POM Switch |
| Keycap | LSA double-shot PBT keycaps, non-shine-through |
| Connectivity | 2.4 GHz / Bluetooth 5.2 / Type-C Wired |
| Polling Rate | 8000 Hz / 2000 Hz / 1000 Hz |
| Firmware | ZMK / Keychron Launcher |
